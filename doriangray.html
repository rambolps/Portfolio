Dorian Gray: The Game



This is a Unity 3D game ported to Unity Engine for HTML5. It is a 2D RPG based on the book, 'The Picture of Dorian Gray' by Oscar Wilde where you play as the main character Dorian Gray. You play through the events in the novel. However, you get to make crucial decisons in the story to change the ending you get! This was created by myself and my friend Justin Chhay

Skills Used
  • C#
  • Tiled Maps
  • Sprite Animation
Technologies Used
  • Unity 3D - Game Engine
  • Firebase Hosting
  • Tiled - Tile Map Editor
